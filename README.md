# 🚀 Niranjan NS — Personal Portfolio

> **Live Site →** [niranjan4560.github.io](https://niranjan4560.github.io)

A modern, single-file personal portfolio website built to showcase my projects, skills, and experience as a Computer Science student at Amrita Vishwa Vidyapeetham.

---

## 👤 About Me

I'm a B.Tech Computer Science student (2023–2027) with hands-on experience in software development, machine learning, and full-stack web technologies. This portfolio replaces my traditional resume with a recruiter-friendly, interactive web experience.

---

## ✨ Features

- ⚡ Fully responsive — works on mobile, tablet, and desktop
- 🎨 Modern dark UI with teal accent and smooth animations
- ⌨️ Typewriter hero effect cycling through roles
- 📊 Animated skill progress bars triggered on scroll
- 🃏 Project cards with tech stacks and key metrics
- 🔗 Clickable contact links — Email, Phone, LinkedIn, WhatsApp
- 🧩 Single `.html` file — zero dependencies, no build step

---

## 🛠️ Built With

| Technology | Purpose |
|---|---|
| HTML5 | Structure & semantics |
| CSS3 | Styling, animations, responsive layout |

No frameworks. No npm. No build tools. Just one clean file.
